What are Tokens?

Tokens are the basic units of text that AI models process. They can be:

  • Words (e.g., "diabetes" = 1 token)
  • Parts of words (e.g., "un" + "happy" = 2 tokens)
  • Punctuation (e.g., "," = 1 token)
  • Special characters (e.g., "üè•" = 1 token)

Why it matters: Token count affects API costs, response length limits, and processing time.
